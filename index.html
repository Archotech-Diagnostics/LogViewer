What to look for?

Focus on Red and Deep Red entries in the Enhanced Log. These indicate where your game is actually failing to execute code. Check the 'Suspect' mod listed immediately above these errors.

Conflicting Mods

Orange messages usually denote two or more mods fighting over the same function. Check if you have multiple mods that try to change the same game mechanic.

Mod Updates

Green: Mod is up to date with the latest Steam Workshop version.
Red: A newer version is available on Steam. Recommendation: Update your mods.
Gray: Unknown or Local mod. No Steam Workshop metadata available for comparison.
